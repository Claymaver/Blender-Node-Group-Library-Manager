\# Changelog



All notable changes to Node Group Library Manager will be documented in this file.



The format is based on \[Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

and this project adheres to \[Semantic Versioning](https://semver.org/spec/v2.0.0.html).



\## \[1.1.0] - 2024-12-16



\### Added

\- \*\*Complete UI Redesign\*\*: Modern, clean interface with better visual hierarchy

\- \*\*Tags System\*\*: Organize node groups with custom tags (e.g., metal, procedural, PBR)

\- \*\*Search Functionality\*\*: Instant search by name or tags

\- \*\*Sort Options\*\*: Sort by name, date, or version count

\- \*\*Custom Storage Path\*\*: Set custom library location in preferences

\- \*\*Export Library\*\*: Export entire library as ZIP for backup or sharing

\- \*\*Import Library\*\*: Import libraries from ZIP files (merges with existing)

\- \*\*Open Library Folder\*\*: Quick access to library directory from preferences

\- \*\*Collapsible Version History\*\*: Expand/collapse version details

\- \*\*Formatted Timestamps\*\*: Human-readable dates and times

\- \*\*Statistics Display\*\*: Shows node group count at a glance

\- \*\*Empty State Guidance\*\*: Helpful instructions when library is empty

\- \*\*Visual Feedback\*\*: Success checkmarks and better status messages



\### Changed

\- \*\*Node Selection\*\*: Now correctly recognizes selected node group nodes, not entire trees

\- \*\*Version Display\*\*: Shows last 5 versions instead of 3 when expanded

\- \*\*UI Layout\*\*: Better spacing, icons, and visual organization

\- \*\*Version Notes\*\*: Displayed more prominently in UI

\- \*\*Error Messages\*\*: More descriptive and user-friendly



\### Fixed

\- Improved handling of node group selection

\- Better error handling for missing files

\- Fixed version tracking for deleted versions



\## \[1.0.0] - 2024-12-10



\### Added

\- Initial release

\- Version control for node groups

\- Save node groups to library with version notes

\- Import node groups from library (latest or specific version)

\- Delete individual versions

\- Delete entire node groups

\- Support for Shader, Compositor, and Geometry nodes

\- N-panel UI in node editors

\- Automatic version incrementing

\- Persistent storage in JSON + .blend files



\### Features

\- Click to add node groups to editor

\- Version history display

\- Confirmation dialogs for deletions

\- Auto-positioning at cursor location

\- Cross-project node group sharing
